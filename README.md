Weather Check POC

This project demonstrates a simple Weather Checking application that allows users to view current weather information for a selected city.

📄 Files Included

->index.html

->style.css

->script.js

✨ Features

1.Search weather by city name

2.Displays temperature, weather condition, and location

3.Real-time weather data using API integration

4.User-friendly interface

5.Dynamic content updates using JavaScript

▶️ How to Run

Open index.html in a web browser and enter a city name to check the weather details.
